# zeiInDaddysHouse
This is a professional private music sharing website for DDP students, and it's also the term project for CMPT 365, SFU.

## How to run?
1. set up Django. See [https://kingstonzen.wordpress.com/2016/03/07/how-to-set-up-django/](https://kingstonzen.wordpress.com/2016/03/07/how-to-set-up-django/
)
2. enter the project root directory, type command

        $ python manage.py runserver

3. Go to browser and enter http://localhost:8000/bbjjzl/ to see the webpage
